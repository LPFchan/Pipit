# 3D assets

- **uguisu_placeholder.glb** — Placeholder Uguisu fob model for Agent 6. Generated by `scripts/generate_uguisu_placeholder_glb_minimal.py` (stdlib-only). Contains meshes: `body`, `button`, `led_rgb` for runtime LED and button-depression logic.
- **iOS:** Add `uguisu_placeholder.usdz` to the app bundle so RealityKit can load it. Convert from the GLB with **Reality Converter** (Mac App Store) or, with Xcode CLI tools: `xcrun usdzconvert uguisu_placeholder.glb -o uguisu_placeholder.usdz`, then add the `.usdz` to the iOS target’s Copy Bundle Resources.
