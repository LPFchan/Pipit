# Research

Curated dependency notes, reusable exploration, and context that should outlive a single execution session belong here.

## Naming

Use filenames shaped like:

- `RSH-YYYYMMDD-NNN-short-title.md`

Each memo should include:

- `Opened: YYYY-MM-DD HH-mm-ss KST`
- `Recorded by agent: <agent-id>`

## What Belongs Here

- Cross-repo dependency notes
- Reusable protocol or ecosystem analysis
- Vendor or tooling context worth keeping
- Findings that inform multiple future sessions

## What Does Not Belong Here

- Raw implementation transcripts
- Ad hoc TODO lists
- Durable decisions without rationale separation
- Current project status updates that belong in `STATUS.md`
